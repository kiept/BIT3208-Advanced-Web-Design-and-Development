## Class Activity 1 — Build a Registration System

Requirements:
- Collect Full Name, Email, Password.
- Store users securely (hashed passwords).

Implementation pointers:
- Use `register.php` as starter code.
- Validate inputs server-side and client-side (`js/validation.js`).
- Use prepared statements and `password_hash()`.

Deliverables:
- Screenshot of the registration form and success message.
- SQL export showing inserted user.
