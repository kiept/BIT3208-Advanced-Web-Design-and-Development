# CakeCraft Responsive Testing

## Responsive design checks

1. Open `index.html` in a browser.
2. Test viewport widths:
   - 360px (mobile)
   - 768px (tablet)
   - 1200px (desktop)
3. Confirm:
   - Menu button appears and toggles navigation on mobile.
   - Hero section stacks vertically on small screens.
   - Product, service, and contact sections become single-column.

## Notes
- Navigation is hidden behind the `Menu` button on screens <= 720px.
- The grid layout collapses to a vertical layout on smaller screens.
- Buttons remain readable and touch-friendly.
