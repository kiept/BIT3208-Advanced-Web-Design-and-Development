# CakeCraft — Week 8 Responsive Website

This folder contains a complete responsive website for a Cake Training and Supplies business.

## Pages
- `index.html` — Home, About, Products, Training Services, Contact
- `styles.css` — mobile-first CSS using Flexbox and CSS Grid
- `script.js` — responsive mobile navigation

## Features
- Mobile-first responsive design
- Responsive navigation menu
- Grid-based product and service sections
- Contact section with form layout
- Works on mobile, tablet, and desktop

## Run
Open `index.html` in a browser, or serve locally:

```bash
cd week8/cake_site
python3 -m http.server 8080
```

Open http://127.0.0.1:8080
