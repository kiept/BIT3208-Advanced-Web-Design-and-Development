# E-store

Smart GameZone is a simple PHP/MySQL web store built to match weekly project requirements for BIT3208.

## Features
- Product catalog with search support
- Product detail pages and cart workflow
- Checkout with order persistence
- User registration and login
- Order history for logged-in customers
- Weekly progress documentation pages

## Setup
1. Place the `smart_gamezone` folder in your PHP server root.
2. Import `smart_gamezone/database.sql` into MySQL.
3. Update `smart_gamezone/includes/db.php` with your database credentials.
4. Run with PHP built-in server:
	```bash
	cd smart_gamezone
	php -S localhost:8000
	```
